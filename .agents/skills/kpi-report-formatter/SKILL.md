# KPI Report Formatter — Advertising Performance Report Standards

Standardized formatting rules for advertising KPI reports. Ensures consistent, readable, actionable output across campaign, ad group, and keyword levels.

## Core Formatting Principles

1. **Scannable First**: Executives should grasp performance in under 30 seconds
2. **Traffic Light System**: 🟢 (good) / 🟡 (watch) / 🔴 (action required)
3. **Period Comparison**: Always show current vs. previous period with Δ change
4. **Anomaly Visibility**: Flag outliers prominently before detail tables
