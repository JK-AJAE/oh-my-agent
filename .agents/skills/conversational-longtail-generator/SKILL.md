# 💬 conversational-longtail-generator

단순 키워드(예: "제주도 카페")를 넘어, 사용자의 구체적인 상황과 질문(예: "제주도 동쪽에서 아이랑 가기 좋고 주차 편한 카페 알려줘")을 공략하는 롱테일 키워드를 생성합니다.

## 🎯 주요 기능

1. **의도 기반 키워드 확장**: 핵심 키워드에 '상황', '대상', '필수 조건'을 조합하여 1,000개 이상의 롱테일 조합 생성.
2. **질문형 키워드 발굴**: 지식iN, 블로그 댓글 등에서 실제 사용자가 묻는 질문 패턴을 분석하여 입찰 키워드화.
3. **저비용 고효율 타겟팅**: 경쟁이 치열한 메인 키워드 대신, 경쟁은 낮고 구매 전환율은 높은 세부 키워드 세트 구축.

## 🛠️ 실행 프로토콜

1. 시드 키워드(Seed Keyword)를 입력받는다.
2. 사용자의 페르소나(나이, 성별, 목적)를 조합하여 질문 시나리오를 짠다.
3. "X를 위한 Y용 Z" 패턴으로 키워드를 무한 확장한다.
4. 확장된 키워드 중 실제 검색량이 있거나 검색 가능성이 높은 항목만 선별하여 비딩 리스트를 생성한다.
